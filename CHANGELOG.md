# Changelog

## 0.0.3

- Improves Marketplace metadata and README presentation.
- Adds a before/after IntelliSense ordering image.

## 0.0.2

- Adds the React Props First extension icon.
- Adds GitHub Actions CI for formatting, linting, language service tests, and VS Code smoke tests.
- Adds completion coverage for `React.forwardRef` and `React.memo` components.

## 0.0.1

Initial release of React Props First.

- Prioritizes custom React component props before inherited DOM and ARIA props in JSX/TSX completions.
- Supports TSX components with typed props.
- Supports JSX when TypeScript can infer props from JSDoc.
- Bundles a TypeScript server plugin inside the VS Code extension.
